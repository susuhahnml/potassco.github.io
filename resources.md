## Documentation

- For clingo series 6, the interactive [Potassco Guide](/guide/) lets you try clingo online and provides a quick introduction to clingo's input language, modeling examples, and usage.
- A comprehensive documentation of our software (and older clingo releases) can be found in the [Potassco PDF Guide](https://github.com/potassco/guide/releases/)

## Teaching

- We have a dedicated webpage with a lot of [teaching material](https://teaching.potassco.org) for Answer Set Programming. This includes lecture slides, exercises, and solutions for courses on ASP and related topics.
- You find our teaching videos on the [Potassco Youtube channel](https://www.youtube.com/channel/UCnvoHDf9RqBJxKPSGdToLzA/feed).

## Other prominent ASP systems

- [DLV](https://dlv.demacs.unical.it/) is an ASP system offering front-ends for several advanced KR formalisms.

- [Wasp](http://alviano.github.io/wasp/) is an ASP solver for propositional logic programs.
