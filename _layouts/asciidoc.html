Edit Clingo Command Line Options

--configuration=<arg>
Set default configuration [auto].
arg: auto, frumpy, jumpy, tweety, handy, crafty, trendy, many, <file>
  • auto: Select configuration based on problem type
  • frumpy: Conservative defaults
  • jumpy: Aggressive defaults
  • tweety: Defaults for ASP problems
  • handy: Defaults for large problems
  • crafty: Defaults for crafted problems
  • trendy: Defaults for industrial problems
  • many: Portfolio configuration
  • <file>: Use configuration file
-s, --stats[=<n>]
Enable statistics: 1=basic, 2=full
-e, --enum-mode <arg>
Configure enumeration algorithm [auto].
arg: bt, record, brave, cautious, auto
  • bt: Backtrack decision literals
  • record: Add nogoods for solutions
  • domRec: Add nogoods over domain atoms
  • brave: Brave consequences (union)
  • cautious: Cautious consequences (intersection)
  • auto: bt for enumeration, record for optimization
--project[=<arg>|no]
Enable projective solution enumeration.
arg: show, project, auto [opts: 0..7] (Implicit: auto,3)
  • Project to atoms in show/project directives or auto-select
  • opts:
    • 1: Use var activity (bt only)
    • 2: Progress saving (bt only)
    • 4: Include aux atoms
-n, --models <n>
Compute at most n models (0 for all)
--opt-mode=<arg>
Configure optimization algorithm.
mode: opt, enum, optN, ignore
  • opt: Find optimal model
  • enum: Models with costs ≤ bound
  • optN: Optimum, then enumerate optimal models
  • ignore: Ignore optimize statements
bound: Set initial bound for objective functions
--opt-stop=<bound>...
Stop optimization on model with cost ≤ bound
--imin=<n>
Minimum steps for incmode
--imax={none|<n>}
Maximum steps for incmode
--istop=<arg>
Stop when: none, sat, unsat, unknown (incmode)
--profile[=off|<detail>[,<type>]]
Enable grounding profiling.
detail: detailed, compact [detailed]
type: step, accu, both [accu]
  • detailed: Detailed info
  • compact: Compact info
  • step: Per step
  • accu: Accumulated [default]
  • both: Both types
-h, --help[=<n>]
Print help: 1=basic, 2=more, 3=full and exit
-V, --verbose[=<n>]
Set verbosity level
-v, --version
Print version and exit
-q, --quiet[=<levels>]
Configure printing of models, costs, calls.
levels: mod, cost, call
  • mod: 0=all, 1=last, 2=no models
  • cost: 0=all, 1=last, 2=no optimize values
  • call: 0=all, 1=last, 2=no call steps [2]
--preprocess[=<arg>]
Print simplified program and exit.
fmt: aspif, smodels, reify [aspif]
  • aspif: ASP intermediate format
  • smodels: Smodels format
  • reify: Reified facts
    • steps: Add step numbers
    • sccs: Print SCCs
--mode=<arg>
Run in: parse, rewrite, ground, solve, clasp mode
