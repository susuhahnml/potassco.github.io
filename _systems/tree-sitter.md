A collection of tree-sitter grammars for the Clingo language.

- [tree-sitter-clingo](https://github.com/potassco/tree-sitter-clingo) provides the tree-sitter grammar for clingo language.
- [tree-sitter-metasp](https://github.com/potassco/tree-sitter-metasp) provides the tree-sitter grammar for the language extension of [metasp](/metasp/).
